Spider on Tianmao and Taobao
============================

###自动识别存放多个天猫与淘宝网址的字符串,并爬取商品ID、价格、评论数、评分、月销量、总库存等相关信息

----------
##主要功能是处理一大堆包含商品的网址字符串，然后先从中提取出商品ID,再自动识别天猫或淘宝并进行爬取

* 商品ID、价格、评论数、评分、月销量、总库存，

* 注意：getid()函数的正则&aid=(\d{11})需要根据实际情况来修改，有时候是&id=(\d{11})!!
